# Multibrawn App 🏡

אפליקציית ניהול לידים ודיוור — ערדית בראון

## התקנה

```bash
npm install
npm run dev
```

## דפלוי ל-Netlify

```bash
npm run build
```
גרור את תיקיית `dist` ל-Netlify או חבר GitHub.

## קבצים

- `src/App.jsx` — האפליקציה הראשית
- `netlify.toml` — הגדרות Netlify
- `vite.config.js` — הגדרות Vite
